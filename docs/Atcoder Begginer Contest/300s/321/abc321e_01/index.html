E - Complete Binary Tree

URL

方針​

まずは、頂点番号 xxx を根とした部分木の距離 kkk の頂点の個数を加算する。

続いて、一つ右隣の頂点 x′x'x′ を根とした部分木における、根から距離 k−2k-2k−2 の頂点の個数を加算する。（※）

その後、「完全二分木の根 111 に到達、もしくは、距離 KKK 分だけ移動する」まで、1つ上の階層に移動し、その度に（※）を実行する。

ちょうど距離 KKK 分だけ移動できた場合は、111 を加算する。

具体例​

与えられたテストケースを N=20, X=4, K=4N=20, \ X=4, \ K=4N=20, X=4, K=4 とする。

まずは、頂点番号 444 を根とした部分木における、根から距離 444 の頂点の個数 000 を加算する。

続いて、一つ右隣の頂点 555 を根とした部分木における、根から距離 222 の頂点の個数 111 を加算する。

111 つ上の階層の頂点 222 に移動し、一つ右隣の頂点 333 を根とした部分木の距離 111 の頂点の個数 222 を加算する。

111 つ上の階層の頂点 111 に移動する。これ以上移動できないため、結果 333 を出力する。

根 xxx からの距離 kkk の頂点の個数の求め方​

根 xxx から距離 kkk の階層に存在する頂点の最小値を LLL 、最大値を RRR とする。NNN はテストケースで与えられる完全二分木の頂点の個数である。

C#
public static long Count(long max, long current, long cnt)
{
    long left = current, right = current;

    while (right < max && 1L <= cnt)
    {
        cnt--;

        left = left * 2L;
        right = right * 2L + 1L;
    }

    if (cnt == 0 && left <= max)
        return Math.Min(max, right) - left + 1L;
    else
        return 0L;
}
